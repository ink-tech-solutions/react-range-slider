# React Range Slider

Range Slider for React

## Installation

```sh
npm install @inkdev/react-range-slider
```

## Usage

```js
import { RangeSlider } from '@inkdev/react-range-slider';

// ...

const [value, setValue] = useState(5);

<RangeSlider min={5} max={14} value={value} setValue={setValue} />;
```
